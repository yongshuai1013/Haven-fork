• The terminal no longer mistakes filenames for web links. Text such as nginx.conf, php.ini, Makefile.in, main.cc or a line of a Java stack trace was underlined and tappable, and tapping it opened a browser at an invented address like https://nginx.co. A detected link must now end where the word ends, and .in, .cc and .app are no longer read as domain endings. Real links still work, bare ones like google.com included.
